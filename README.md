+ ### Structural Patttern
  > 구조를 유연하고 효율적으로 유지하면서 클래스나 객체를 조합해 더 큰 구조를 만드는 패턴.  
  > 객체를 어떻게 조합할 것인가를 정의해놓은 패턴. 어떤 새로운 기능을 제공하기위해서, 객체를 조합하는 방법들을 패턴화 시킨 것.
